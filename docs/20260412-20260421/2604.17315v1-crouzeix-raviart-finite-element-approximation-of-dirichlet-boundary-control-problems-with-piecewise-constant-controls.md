## 摘要
本文研究由 Poisson 方程支配的 Dirichlet 边界控制问题，其中控制变量是定义在二维有界凸多边形域边界上的平方可积函数。采用超弱形式，并利用 Crouzeix-Raviart 有限元对状态变量进行离散化，同时对控制变量采用分片常数函数进行离散化。此外，文中建立了控制变量的最优阶先验误差估计。

## Abstract
This article examines the Dirichlet boundary control problem governed by the Poisson equation, where the control variables are square integrable functions defined on the boundary of a two-dimensional bounded, convex, polygonal domain. It employs an ultra-weak formulation and utilizes Crouzeix-Raviart finite elements to discretize the state variable, while employing piecewise constants for the control variable discretization. Furthermore, it establishes an optimal order a priori error estimate for the control variable.

## 二、方法论与技术路线

### 1. 核心思想

- 通过 **超弱形式法（method of transposition）**，将状态方程重新解释为：
  \[
  (y, \Delta \phi) = \langle u, \frac{\partial \phi}{\partial n} \rangle, \quad \forall \phi \in X
  \]
  其中 \(X = \{ \phi \in H^1_0(\Omega) : \Delta \phi \in L^2(\Omega) \}\)。

- 控制变量 \(u\) 定义在边界上，属于 \(L^2(\Gamma)\)；状态变量 \(y\) 属于 \(L^2(\Omega)\)。

### 2. 离散化策略

- **状态和伴随状态离散化**：  
  使用 **一阶非一致 Crouzeix-Raviart 有限元空间 (CR space)**，该空间的自由度定义在单元的边中点，使其自然契合分片常数边界控制的结构。  

- **控制变量离散化**：  
  控制空间 \(U_h\) 由边上的分片常数函数组成，即每个边上取一个常数控制。

- **离散最优控制系统**：  
  定义离散状态方程和伴随方程：
  \[
  a_h(y_h, v_h) = 0, \quad \forall v_h \in V_h^0
  \]
  \[
  a_h(p_h, v_h) = (y_h - y_d, v_h)
  \]
  边界条件通过边积分匹配实现：
  \[
  \int_e y_h ds = \int_e u_h ds, \quad \forall e \in E_b^h
  \]

- **控制到状态映射算子**：
  \[
  S_h : U_h \rightarrow V_h
  \]
  并进一步设计其对偶算子 \(S_h^*\) 以求取最优性系统。

### 3. 误差分析与关键结论

作者推导了多项理论估计：

- 边界提升函数（boundary lifting）在 \(L^1(\Omega)\) 范数下具有稳定性：  
  \[
  \| \tilde{w}_h \|_{L^1(\Omega)} \leq C h \| u_h \|_{L^1(\Gamma)}
  \]

- 连续与离散映射之间的误差：
  \[
  \| S u - S_h u_h \| \leq Ch^{1 - \epsilon}
  \]
  为几乎线性收敛率。

- 最终最优控制误差估计：
  \[
  \| \bar{u} - \bar{u}_h \|_{L^2(\Gamma)} \leq C h^{1/2 - \epsilon}
  \]
  即达到了 **近最优收敛阶（almost optimal order）**。
