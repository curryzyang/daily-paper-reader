## 摘要
油藏模拟工作流程面临一个根本的数据不对称问题：输入参数场（地统计渗透率实现、孔隙度分布）可以任意生成大量数据，而现有的神经算子替代模型仍依赖于庞大且昂贵的、有标签的模拟轨迹数据集，无法利用这些无标签结构。我们提出了\textbf{PI-JEPA}（Physics-Informed Joint Embedding Predictive Architecture，物理约束联合嵌入预测架构），这是一种替代模型预训练框架，可在\emph{完全无需进行 PDE 求解}的情况下，通过对无标签参数场进行掩码潜变量预测，并在每个子算子的 PDE 残差上施加正则化来训练。预测器网络的结构与主方程的 Lie–Trotter 算子分裂分解相协调，为每个子过程（压力、饱和度传输、反应）单独分配一个受物理约束的潜变量模块，使得在仅使用约 100 次有标签模拟运行的情况下即可进行微调。在单相 Darcy 流动中，PI-JEPA 在 $N_\ell{=}100$ 时的误差比 FNO 低 $1.9\times$，比 DeepONet 低 $2.4\times$，在 $N_\ell{=}500$ 时相比仅监督训练提升了 24\%，表明无标签替代模型预训练显著减少了多物理场替代模型部署所需的模拟预算。

## Abstract
Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

## 一、核心问题与研究背景

- **背景与动机**  
  多物理场耦合的偏微分方程（PDE）仿真广泛应用于储层工程、地质 CO₂ 存储、污染物迁移等场景。  
  然而，该类高保真数值模拟需要大量计算资源，每次运行耗时数小时至数天，使得建立监督式神经算子（如 FNO、DeepONet）替代模型昂贵且难以扩展。  

- **问题本质**  
  存在 **数据不对称**：输入参数场（如渗透率、孔隙度）可以任意生成，而有标签仿真数据（完整 PDE 求解结果）却极其稀缺。  
  传统神经算子无法利用这种无标签结构，导致数据效率极低。  

- **目标与意义**  
  作者旨在**设计一种可利用无标签参数场进行预训练的物理约束替代模型框架**，使得在只有很少有标签仿真样本时仍能获得高精度模型，从根本上降低仿真预算。

## 二、方法论与技术方案

### （1）总体思想
提出 **PI-JEPA（Physics-Informed Joint Embedding Predictive Architecture）**，借鉴了自监督学习中的 **JEPA** 思路，将其引入物理仿真替代建模领域，实现：
- **完全无需 PDE 求解的预训练阶段**；
- 在潜空间中利用掩码预测任务进行表示学习；
- 每个物理子过程（算子分裂）均有独立的物理约束模块。

### （2）核心结构与原理

- **架构组成**
  - **Context Encoder（上下文编码器）** \(f_\theta\)：输入未掩码的空间区域。
  - **Target Encoder（目标编码器）** \(f_\xi\)：EMA（指数移动平均）更新的稳定目标网络。
  - **预测器组** \(\{g_{\phi_k}\}_{k=1}^K\)：与 PDE 求解过程中的算子分裂结构对齐（如压力求解、饱和度传输、化学反应三个子算子）。

- **训练目标公式**
  \[
  L = L_\text{pred} + \lambda_p \sum_{k=1}^{K} L_\text{phys}^{(k)} + \lambda_r L_\text{reg}
  \]
  其中：
  - \(L_\text{pred}\)：潜空间掩码预测误差；
  - \(L_\text{phys}\)：每个算子级别的 PDE 残差约束（保证物理一致性）；
  - \(L_\text{reg}\)：VICReg 变异-协方差正则，用于防止潜空间塌陷。

- **物理算子分裂对应**
  - \(L_1\)：压力子算子（椭圆方程）；
  - \(L_2\)：饱和度传输（双相流）；
  - \(L_3\)：化学反应（代数一致条件）。

- **掩码策略**
  - 采用**时空块掩码**：当前时刻的部分空间区域作为上下文，下一个时间步的邻域区域作为预测目标；
  - 学习“从上下文到未来”的潜变量演化规律。

- **推理模式**
  - 微调后模型以自回归方式在潜空间中滚动预测；
  - 可在每步利用解码器恢复物理场，实现闭环预测。

## 六、主要结论与发现

- **性能提升**
  - 在单相 Darcy 流（64×64）中，PI-JEPA 在 \(N_\ell=100\) 时：
    - 错误率比 FNO 低 **1.9×**；
    - 比 DeepONet 低 **2.4×**；
    - 相比从零训练能提升 **16–24%**。
  - 两相 CO₂–水流任务中提升更大（约 20–25%）。
  - 反应传输任务（ADR）中提升中等（1–2%），原因是预训练域差异。

- **理论发现**
  - 给出样本复杂度分析：算子分裂预训练将微调样本需求从 \(O(n^2ε^{-2})\) 降到 \(O(d^2Kε^{-2})\)，即随潜维度和分裂子数显著降低。

- **应用意义**
  - 无需昂贵仿真即可学习物理一致的替代模型；
  - 使油藏工程师仅需数十次仿真即可得到高精度代理。
