## 摘要
储层模拟工作流程面临一种根本性的数据不对称：输入参数场（地质统计渗透率实现、孔隙度分布）可以任意生成，但现有的神经算子替代模型需要大量昂贵的有标签模拟轨迹，无法利用这些无标签结构。我们提出了\textbf{PI-JEPA}（Physics-Informed Joint Embedding Predictive Architecture，物理约束联合嵌入预测架构），一种替代模型的预训练框架，在完全不进行 PDE 求解的情况下，通过在无标签参数场上进行掩码潜空间预测并施加每个子算子 PDE 残差的正则化进行训练。预测器组的结构与控制方程的 Lie–Trotter 算子分裂分解相一致，为每个子过程（压力、饱和度传输、反应）分别分配一个物理约束的潜层模块，从而使得仅需约 100 次有标签模拟运行即可进行微调。在单相 Darcy 流动案例中，PI-JEPA 在 $N_\ell{=}100$ 时的误差比 FNO 低 $1.9\times$，比 DeepONet 低 $2.4\times$，在 $N_\ell{=}500$ 时较纯监督训练提升 24\%，表明无标签替代预训练能显著减少多物理场替代模型部署所需的模拟预算。

## Abstract
Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

## 一、核心问题与研究背景

- **研究动机**  
  多物理场仿真（如地下流体流动、CO₂ 存储、反应传输）依赖复杂 PDE 数值求解，单次高保真模拟常需数小时至数天，成本极高。  
  当前神经算子（如 FNO、DeepONet）可用作加速替代模型，但依赖大量昂贵的带标签模拟样本。  
  现实中存在显著**数据不对称性**：  
  - 无标签输入参数场（例如渗透率场、孔隙度分布）可廉价、大量生成；  
  - 但带模拟结果的完整轨迹生成代价巨大。  
  因此，核心问题是**如何无需昂贵标签，利用这些“免费”的无标签参数场进行物理一致的预训练**，从而显著降低监督样本需求。

## 二、方法论：PI-JEPA 框架

### 1. 核心思想

- 借鉴 LeCun 提出的 **Joint Embedding Predictive Architecture (JEPA)** 的自监督理念。  
- 将 **物理信息约束（Physics-Informed）** 与 **JEPA 潜空间掩码预测** 相结合，形成 **PI-JEPA（Physics-Informed JEPA）**。
- 关键思路：  
  1. 在不进行任何 PDE 求解的情况下，使用**未标注参数场**做掩码潜变量预测任务；  
  2. 以 PDE 的**算子分裂结构（operator splitting）**为指导，为每个物理子过程设计独立的预测器并施加物理残差约束；  
  3. 通过自监督预训练学得物理一致的潜空间编码，后续仅需少量标注样本微调。

### 2. 模型结构（见 Figure 1）

- **编码器对 (fθ, fξ)**：  
  - fθ 为上下文编码器（context encoder），fξ 为其 EMA（指数滑动平均）目标编码器；  
  - 采用 *Fourier 增强的 Transformer 编码器*（Fourier + Attention 架构）。  
- **潜空间预测器组 {gϕ₁,…,gϕK}**：  
  - 每个 gϕₖ 对应 PDE 的一个子算子；  
  - 如 Darcy 方程含压强算子与饱和度传输算子（K=2），反应传输问题则含第三个反应算子（K=3）。  
- 预测任务：掩码目标区域潜编码的预测  
  \[
  L_{pred} = \| \hat{z}_t - \text{sg}(f_\xi(u_t)) \|^2_2
  \]
- **总损失函数**  
  \[
  L = L_{pred} + \lambda_p \sum_k L^{(k)}_{phys} + \lambda_r L_{VICReg}
  \]
  - \(L^{(k)}_{phys}\)：每个子算子的 PDE 残差正则项；  
  - \(L_{VICReg}\)：方差–协方差正则，防止潜空间塌陷。  

### 3. 算法流程

1. **无标签预训练阶段（Pretrain）**  
   - 输入：大量未标注渗透率场或中间态场；  
   - 掩码潜变量预测 + 各子算子 PDE 残差 + VICReg 正则；  
   - 不需运行模拟器。  
2. **少量标注微调阶段（Finetune）**  
   - 使用约 10–500 条完整模拟轨迹对预训练编码器进行微调；  
   - 输出预测头接在预训练编码器上，恢复到物理场分布。

## 六、主要结论与发现

1. **显著数据效率提升**  
   - 在单相 Darcy 流中，PI-JEPA 在 \(N_\ell=100\) 时误差比 FNO 低 1.8×、比 DeepONet 低 2.2×；  
   - 两相 CO₂–水流中，PI-JEPA 相比 FNO 误差下降 2.7×；  
   - ADR 系统中，误差比 FNO 低 1.9×。  
2. **预训练带来 6–14% 额外收益**，尤其在 \(N_\ell≥100\) 时。  
3. **消融结果**  
   - 去除算子分裂 → 性能下降 16%；  
   - 去除 VICReg → 性能下降 17.7%；  
   - 去除物理残差略提升（–8.1%），说明该项在当前实现下未正向贡献。  
4. **理论分析**  
   - 给出样本复杂度证明：算子分裂后 fine-tuning 样本复杂度从 \(O(n^2\varepsilon^{-2})\) 降至 \(O(d^2K\varepsilon^{-2})\)。  
5. **领域迁移性**  
   - Darcy 预训练在两相流任务上仍优于 scratch 模型；  
   - ADR 任务因域差较大，需领域匹配预训练才具优势。
