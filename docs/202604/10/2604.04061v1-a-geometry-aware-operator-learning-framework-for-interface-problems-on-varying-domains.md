## 摘要
在变化域上求解偏微分方程（PDE）界面问题是设计与优化中的一个关键任务，但对于传统求解器而言，其计算代价依然高昂。尽管算子学习在固定几何结构上已展现出潜力，但其在几何相关界面问题中的应用仍然鲜有探索。为弥补这一空白，我们提出了一种基于扩展的神经算子框架，可适用于一般线性界面问题。我们方法的核心创新在于将定制有限点法（Tailored Finite Point Method, TFPM）与基础网络相结合，从而减少内存消耗，并有效缓解维度灾难。在理论方面，我们建立了Helmholtz算子关于域扰动的连续性，并为所提出的编码提供了严格的误差估计。大量数值实验表明，我们的框架在精度和鲁棒性方面均达到了当前最优水平。因此，该工作为变化域仿真提供了一种高效、数据友好的工具，为实时形状优化开辟了新的可能性。

## Abstract
Solving Partial Differential Equation (PDE) interface problems on varying domains is a critical task in design and optimization, yet it remains computationally prohibitive for traditional solvers. Although operator learning has shown promise on fixed geometries, its potential for geometry-dependent interface problems has been largely unexplored. To bridge this gap, we propose an extension-based neural operator framework applicable to general linear interface problems. A key innovation of our method is the integration of the Tailored Finite Point Method (TFPM) with our base network, which reduces memory consumption and effectively alleviates the curse of dimensionality. On the theoretical front, we establish the continuity of the Helmholtz operator with respect to domain perturbations and provide rigorous error estimates for the proposed encodings. Comprehensive numerical experiments demonstrate that our framework achieves state-of-the-art accuracy and robustness. Consequently, this work provides a powerful, data-efficient tool for varying-domain simulations, offering new possibilities for real-time shape optimization.

## 二、方法论

### 2.1 核心思想

- 采用**扩展型（extension-based）算子学习方法**：  
  将所有变化的子域嵌入到一个更大的固定矩形域 \([s,t]^d\)，通过“零扩展”将定义在不同域上的函数映射到该固定域上，使得神经网络可以在统一空间中学习算子。
- 设计了**几何感知（geometry-aware）编码**，用以描述不同几何域和界面；同时结合**Tailored Finite Point Method (TFPM)** 以改进数值稳定性与内存效率。

### 2.2 关键技术与结构

1. **输入-输出映射定义**  
   以Helmholtz型方程为代表问题：
   \[
   -\nabla(a(x)\nabla u)+b(x)u=f(x), \quad \text{with interface jumps }[u], [a\nabla u\cdot n].
   \]  
   学习几何与物理参数到解的算子 \(G\)：  
   \[
   G:(a,b,f,g,\alpha,\beta) \mapsto u.
   \]
   通过零扩展与几何编码得到统一的扩展算子 \(\hat{G}\)。

2. **几何编码方式**
   - **特征函数编码**（Characteristic Function）：简单，内存小，但精度有限。
   - **符号距离函数编码**（Signed Distance Function, SDF）：精度更高，但在多界面情况下开销更大。
   - 理论分析表明：特征函数的编码误差阶为 \(O(n^{-1/2})\)，SDF 误差阶为 \(O(n^{-3/2})\)。

3. **神经网络结构**
   - 基于 **Fourier Neural Operator (FNO)** 架构：
     \[
     G_\theta = Q \circ (W_L+K_L+b_L) \circ ... \circ P,
     \]
     其中 \(K_L\) 是傅里叶卷积算子，P、Q 为编码与解码器。
   - 采用多层傅里叶变换层进行全局感受域学习。

4. **TFPM 基函数嵌入**
   - 替代常规有限元基，使用TFPM的局部解析基（指数型函数）表示局部解，提高表达能力与抗维度爆炸性。
   - 网络预测TFPM系数，然后通过重建算子 \(Re(c)\) 得到 \(u(x)\)。
   - 优点：减少GPU显存需求，适配高维问题。

5. **损失函数**
   \[
   L(\theta)=\frac{1}{N}\sum_i \| \hat{G}_\theta - \hat{u}\|^2_{L^2([s,t]^d)},
   \]
   若使用TFPM基，则损失中先通过重建算子恢复 \(u\)。
