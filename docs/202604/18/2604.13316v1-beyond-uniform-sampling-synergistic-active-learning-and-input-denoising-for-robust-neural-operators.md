## 摘要
神经算子已成为物理仿真的快速替代模型，但它们仍然对对抗性扰动极为敏感，这在安全关键的数字孪生部署中构成了严重隐患。我们提出了一种协同防御方法，将基于主动学习的数据生成与输入去噪架构相结合。主动学习部分利用差分进化攻击自适应地探测模型弱点，并在已发现的脆弱位置生成针对性的训练数据，同时通过自适应平滑比率保护机制保持基线精度。输入去噪部分在算子架构中引入可学习的瓶颈，以过滤对抗性噪声，同时保留与物理相关的特征。在黏性 Burgers 方程基准测试中，该联合方法实现了 2.04% 的综合误差（1.21% 基线误差 + 0.83% 鲁棒性误差），相较于标准训练（15.42% 综合误差）减少了 87%，并优于仅使用主动学习（3.42%）或仅使用输入去噪（5.22%）的结果。更广泛来看，我们的结果结合先前工作的跨架构脆弱性分析表明，神经算子的最优训练数据依赖于架构：由于不同架构的敏感性集中在不同的输入子空间，均匀采样无法充分覆盖所有模型的脆弱性分布。这些发现对神经算子在包括核反应堆监测在内的安全关键能源系统中的部署具有潜在意义。

## Abstract
Neural operators have emerged as fast surrogate models for physics simulations, yet they remain acutely vulnerable to adversarial perturbations, a critical liability for safety-critical digital twin deployments. We present a synergistic defense that combines active learning-based data generation with an input denoising architecture. The active learning component adaptively probes model weaknesses using differential evolution attacks, then generates targeted training data at discovered vulnerability locations while an adaptive smooth-ratio safeguard preserves baseline accuracy. The input denoising component augments the operator architecture with a learnable bottleneck that filters adversarial noise while retaining physics-relevant features. On the viscous Burgers' equation benchmark, the combined approach achieves a 2.04% combined error (1.21% baseline + 0.83% robustness), representing an 87% reduction relative to standard training (15.42% combined) and outperforming both active learning alone (3.42%) and input denoising alone (5.22%). More broadly, our results, combined with cross-architecture vulnerability analysis from prior work, suggest that optimal training data for neural operators is architecture-dependent: because different architectures concentrate sensitivity in distinct input subspaces, uniform sampling cannot adequately cover the vulnerability landscape of all models. These findings have potential implications for the deployment of neural operators in safety-critical energy systems including nuclear reactor monitoring.

- **研究问题**：神经算子（Neural Operators）是一类能够学习函数空间映射的强大工具，用于求解偏微分方程（PDE），广泛应用于工程仿真和数字孪生系统。然而，此类模型对输入的微小扰动（尤其是对抗性扰动）极为敏感，即使扰动在物理意义上很微弱，也可能造成预测结果与真实物理解严重偏离。

## 二、方法论：核心思想与关键技术

### 1. 整体框架

论文提出了一个**协同鲁棒防御框架**，结合：
- **主动学习（Active Learning）**：自适应数据生成；
- **输入去噪（Input Denoising）**：结构层面的防御。

两者相互补充，既在训练数据层面增强模型认知，又在模型结构层面提升对噪声的抵御力。

### 2. 主动学习机制（数据层防御）

- **目标**：识别并强化模型在脆弱输入处的表现。
- **流程（算法1）**：
  1. 初始化一批平滑初始条件样本进行基础训练；
  2. 使用**差分进化（Differential Evolution, DE）攻击**探测模型脆弱区域；
  3. 对这些脆弱点生成新的训练数据，并从真实物理数值求解器中获得对应标签；
  4. 自适应调整“平滑比率参数”α，使得新增数据既包含脆弱样本又保留部分平滑样本；
  5. 迭代执行上述步骤直到耗尽总模拟预算。
- **关键特性**：与传统对抗训练不同，该方法不强制模型抵抗扰动，而是教模型正确地响应受扰动的物理输入，从而保持“对物理一致”的鲁棒性。

### 3. 输入去噪架构（结构层防御）

- **核心结构**：在 DeepONet 的分支网络前加入一个**可学习的自编码器瓶颈**：
  \[
  \tilde{u}_0 = w \cdot D(u_0) + (1 - w) \cdot u_0
  \]
  其中 `D`为编码-解码模块（64→32→64），`w`为可学习的权重，用于平衡去噪强度与信息保留。
- **功能**：通过压缩输入特征空间，降低对高频噪声的敏感度，同时保持与物理相关的低频结构。
- **训练模式**：端到端联合优化，使去噪瓶颈学习到与算子输出误差互补的特征抑制模式。

### 4. 协同策略（Combined Defense）

- 数据层主动学习强化了模型的物理映射能力；
- 结构层去噪提升了输入鲁棒性；
- 最终组合（AL + Denoising）在两个维度同时取得显著性能提升。

## 三、实验设计

### 1. 测试场景
- **物理系统**：黏性 Burgers 方程（Viscous Burgers’ Equation）作为基准 PDE。
- **离散配置**：空间点数 nx = 64，黏性系数 ν = 0.1，终止时间 T = 1.0。
- **攻击方式**：高斯型、局部化扰动（m ∈ [−0.3, 0.3]，σ = 5），模拟传感器噪声或校准漂移。

### 2. 对比方法
共比较五种策略：
1. **Baseline**：标准平滑数据训练。
2. **Balanced**：混合平滑和随机扰动数据（60% perturbed / 40% smooth）。
3. **Denoising only**：仅使用去噪架构但无主动学习。
4. **Active Learning (AL)**：采用主动学习机制但无去噪结构。
5. **AL + Denoising**：联合使用两种防御。

### 3. 评估指标
- **Baseline Error**：干净数据上的相对 L2 误差；
- **Robustness Error**：受扰动数据的预测与物理解之间的误差；
- **Combined Score**：二者之和，用于同时评估准确度与鲁棒性。
