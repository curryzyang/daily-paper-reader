## 摘要
本文研究由 Poisson 方程支配的 Dirichlet 边界控制问题，其中控制变量是定义在二维有界、凸、多边形域边界上的平方可积函数。本文采用超弱形式，并使用 Crouzeix-Raviart 有限元对状态变量进行离散化，而对控制变量采用分片常数离散化。此外，本文为控制变量建立了最优阶的先验误差估计。

## Abstract
This article examines the Dirichlet boundary control problem governed by the Poisson equation, where the control variables are square integrable functions defined on the boundary of a two-dimensional bounded, convex, polygonal domain. It employs an ultra-weak formulation and utilizes Crouzeix-Raviart finite elements to discretize the state variable, while employing piecewise constants for the control variable discretization. Furthermore, it establishes an optimal order a priori error estimate for the control variable.

## 二、方法论与技术细节

### 1. 总体思路
- 引入 **超弱（ultra-weak）形式** 来重新定义状态方程，使控制 \(u \in L^2(\Gamma)\) 仍能产生唯一的状态解。
- 状态变量采用 **CR 非符合有限元** 离散化，其自由度定义在边的中点，天然与 **分片常数控制** 相匹配。
- 控制变量使用 **逐边常值函数（piecewise constant）** 进行离散化。

### 2. 关键公式与定义
- **超弱形式**：
  \[
  (y, \Delta \phi) = \langle u, \partial \phi / \partial n \rangle, \quad \forall \phi \in X := \{ \phi \in H^1_0(\Omega) : \Delta \phi \in L^2(\Omega) \}.
  \]
- **控制到状态算子** \(S: L^2(\Gamma) \to L^2(\Omega)\)，离散版本 \(S_h: U_h \to V_h\)。
- **离散优化问题**：
  \[
  \min_{u_h \in U_{h,ad}} j_h(u_h) = \frac{1}{2}\| S_hu_h - y_d \|^2 + \frac{\alpha}{2}\|u_h\|^2_{L^2(\Gamma)}.
  \]
- **离散最优性系统** 包含：
  1. 离散状态方程；
  2. 离散伴随方程；
  3. 离散 Karush–Kuhn–Tucker (KKT) 条件。

### 3. 理论推导核心
- 利用 **CR 插值算子 \(I_{CR}\)** 的逼近特性与稳定性：
  \[
  \|v - I_{CR}v\|_{L^2} + h\|v - I_{CR}v\|_h \lesssim h|v|_{H^1}.
  \]
- 建立控制变量在 \(L^2(\Gamma)\) 意义下的 **最优阶先验误差估计**：
  \[
  \| \bar{u} - \bar{u}_h \|_{L^2(\Gamma)} \le Ch^{1/2 - \epsilon}, \quad \forall \epsilon > 0.
  \]
- 分析边界扩展函数（boundary lifting）的稳定性及在域内的局部集中行为。
