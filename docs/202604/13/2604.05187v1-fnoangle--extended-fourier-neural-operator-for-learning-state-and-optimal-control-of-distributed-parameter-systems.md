## 摘要
我们提出了一种扩展的傅里叶神经算子（FNO）架构，用于学习由偏微分方程（PDE）支配的系统的状态和线性二次加性最优控制。利用 Ehrenpreis-Palamodov 基本原理，我们证明了具有常系数的线性 PDE 的任意状态和最优控制都可以表示为复数域中的一个积分。该积分表示中的被积函数包含与逆傅里叶变换中相同的指数项，而后者用于表示 FNO 层中的卷积算子。受此启发，我们通过将逆傅里叶变换中的频率变量从实数域扩展到复数域，修改了 FNO 层，以捕捉基本原理中的积分表示。我们通过非线性 Burgers 方程的状态与最优控制学习实验展示了该方法的性能，与标准 FNO 相比，其训练误差降低了一个数量级，并且在非周期性边界值的预测上更加准确。

## Abstract
We propose an extended Fourier neural operator (FNO) architecture for learning state and linear quadratic additive optimal control of systems governed by partial differential equations. Using the Ehrenpreis-Palamodov fundamental principle, we show that any state and optimal control of linear PDEs with constant coefficients can be represented as an integral in the complex domain. The integrand of this representation involves the same exponential term as in the inverse Fourier transform, where the latter is used to represent the convolution operator in FNO layer. Motivated by this observation, we modify the FNO layer by extending the frequency variable in the inverse Fourier transform from the real to complex domain to capture the integral representation from the fundamental principle. We illustrate the performance of FNO in learning state and optimal control for the nonlinear Burgers' equation, showing order of magnitude improvements in training errors and more accurate predictions of non-periodic boundary values over FNO.

## 一、研究问题与背景

- **研究动机**：  
  分布参数系统（Distributed Parameter Systems, DPS）广泛存在于流体力学、气候建模、工程控制等领域，其动态由偏微分方程（PDE）描述。传统的最优控制方法（如 LQR）在高维情形计算代价巨大。  
  神经算子（Neural Operators）能够直接学习“函数到函数”的映射，为 PDE 的快速求解与控制带来了新思路。其中，**傅里叶神经算子（FNO）**采用卷积在频域表达的方法，能高效逼近 PDE 解算算子，但局限在周期性边界条件，难以准确建模非周期或有界域问题。

- **核心问题**：  
  如何让神经算子突破仅适用于周期性情形的限制，从理论上更好地表示和学习具有**非周期边界**和**复杂最优控制结构**的 PDE 系统？

## 二、方法论：FNO∠θ 的核心思想与实现机制

- **理论基础**：  
  作者基于 **Ehrenpreis–Palamodov 基本原理**，该原理指出线性常系数 PDE 的所有解及对应最优控制可表示为复平面上的积分。该积分中的指数项形式类似逆傅里叶变换，但积分域位于复数空间。

- **核心思想**：  
  将传统 FNO 的逆傅里叶变换从 **实频率域** 扩展到 **复频率域**，引入一个可学习的相位参数θ，使神经算子能表达更广泛的复指数形式，从而捕捉非周期边界及复杂控制解。

- **关键技术细节**：  
  - 传统 FNO 层的线性卷积算子：
    \[
    (K_l v_l)(ξ) = \frac{1}{(2π)^n} ∫_{ℝ^n} e^{i k·ξ} \hat{K}_l(k) \hat{v}_l(k) dk
    \]
  - 扩展算子 FNO∠θ 中的复频率：
    \[
    (K^{θ}_l v_l)(ξ) = \frac{1}{(2π)^n} ∫_{ℝ^n} e^{i z(k,θ_l)·ξ} \hat{K}_l(k) \hat{v}_l(k) dk, \quad z_j(k,θ_l) = k_j e^{iθ_{l,j}}
    \]
  - 当 θ=0 时退化为标准 FNO；当 θ≠0 时，频率变量具有复相位，使算子在有界域上具有更强的表达能力。
  - 所有层仍保留 FNO 的 lifting/projection 结构与非线性激活函数（GELU）。

- **算法流程（概述）**：
  1. **输入提升**（lifting）到高维特征空间；
  2. 多层 FNO∠θ 作用，每层执行复频率卷积；
  3. **投影**到输出空间，得到状态或控制算子；
  4. 采用监督学习最小化预测结果与真实数据均方误差（MSE）。

## 三、实验设计

- **实验任务**：
  1. **学习状态算子**：从 Burgers 方程的边界条件映射到系统状态；
  2. **学习最优控制算子**：从边界条件映射到相应的最优控制输入。

- **选用 PDE 场景**：  
  使用非线性一维 Burgers 方程  
  \[
  ∂_tϕ + ϕ∂_xϕ = ν∂_{xx}ϕ + u
  \]
  在有界域 Ω = [0,1] × [0,0.5] 上仿真，设置粘度 ν=0.05。

- **数据生成**：
  - 初始条件：ϕ₀(x)=sin(πx)；

  - 训练集包含 50 条不同边界条件下的状态轨迹。

- **对比方法**：  
  - 标准傅里叶神经算子（FNO）；
  - 扩展复域傅里叶算子（本文提出的 FNO∠θ）。

- **指标**：

## 六、主要结论与发现
