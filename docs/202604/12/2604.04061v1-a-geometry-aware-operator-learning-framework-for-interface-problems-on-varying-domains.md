## 摘要
在可变域上求解偏微分方程（PDE）界面问题是设计与优化中的关键任务，但对于传统求解器而言计算代价极高。尽管算子学习在固定几何结构上已展示出潜力，但其在依赖几何的界面问题中的应用仍 largely 未被探索。为弥补这一空白，我们提出了一种基于扩展的神经算子框架，可适用于一般线性界面问题。我们方法的关键创新在于将定制有限点法（Tailored Finite Point Method, TFPM）与基础网络相结合，从而降低内存消耗，并有效缓解维度灾难。在理论方面，我们建立了Helmholtz算子关于域扰动的连续性，并为所提出的编码给出了严格的误差估计。大量数值实验表明，我们的框架在精度和鲁棒性方面达到了当前最优水平。因此，该研究为可变域模拟提供了一种高效的数据驱动工具，为实时形状优化开辟了新的可能性。

## Abstract
Solving Partial Differential Equation (PDE) interface problems on varying domains is a critical task in design and optimization, yet it remains computationally prohibitive for traditional solvers. Although operator learning has shown promise on fixed geometries, its potential for geometry-dependent interface problems has been largely unexplored. To bridge this gap, we propose an extension-based neural operator framework applicable to general linear interface problems. A key innovation of our method is the integration of the Tailored Finite Point Method (TFPM) with our base network, which reduces memory consumption and effectively alleviates the curse of dimensionality. On the theoretical front, we establish the continuity of the Helmholtz operator with respect to domain perturbations and provide rigorous error estimates for the proposed encodings. Comprehensive numerical experiments demonstrate that our framework achieves state-of-the-art accuracy and robustness. Consequently, this work provides a powerful, data-efficient tool for varying-domain simulations, offering new possibilities for real-time shape optimization.

## 二、方法论概述

### 1. 核心思想
提出一个**基于函数扩展（Extension-based）的几何感知神经算子框架**，结合**Tailored Finite Point Method (TFPM)** 来提高算子学习在可变域场景中的效率、稳定性与内存利用率。

### 2. 技术组成

- **扩展算子定义**：将原定义在不同域上的函数通过零扩展映射到一个统一的固定大域 \([s,t]^d\)，并在该固定网格上学习一个扩展算子：
  \[
  \hat{G}:\{\varphi_\Omega,\varphi_\Gamma,\hat{a},\hat{b},\hat{f},\hat{g},\hat{\alpha},\hat{\beta}\}\to \hat{u}
  \]
  从而实现多个几何域的统一编码与学习。

- **编码策略**：
  - **特征函数编码**：以 0 / 1 值在网格中表示各域与界面位置。
  - **带符号距离函数 (SDF)** 编码：引入距离信息，提供更高的几何精度。
  - 并对低维流形（如边界 ∂Ω、界面 Γ）采用高斯权重映射的投影编码以保持信息连续性。

- **神经算子架构**：采用 **Fourier Neural Operator (FNO)**，结合频谱卷积层、点云编码器 P、解码器 Q，实现端到端的算子学习。

- **引入 TFPM 基**：
  - 在基础算子输出阶段，不直接预测解 \(u\)，而预测局部参数化基系数 \(c_{ij}\)，由 TFPM 基函数重建解：
    \[
    G_{TFPM}: a,b,f,g,\alpha,\beta \to c,\quad u=R_e(c)
    \]
  - 该基函数在每个子区域采用指数形式 \(e^{\pm \mu x}, e^{\pm \mu y}\)，以更好表示波动与界面行为。
  - **优势**：减少GPU存储需求；提高在高维情况下的学习稳定性。

### 3. 理论分析

- **连续性证明**：通过引用 Costabel 与 Simon 的边界积分形导数理论，证明 Helmholtz 算子在域和界面形变下的连续性。
- **编码误差分析**：
  - 特征函数编码误差阶为 \(O(n^{-1/2})\)
  - SDF 编码误差阶为 \(O(n^{-3/2})\)，更精确但代价更高。
