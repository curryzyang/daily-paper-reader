## 摘要
神经算子方法已成为学习无限维函数空间之间映射的强大工具，但其在最优控制中的潜力尚未被充分探索。本文聚焦于多任务控制问题，其解是从任务描述（例如代价函数或动力学函数）到最优控制规律（例如反馈策略）的映射。我们使用一种置换不变的神经算子架构来逼近这些解算子。在一系列参数化最优控制环境和一个运动基准测试中，通过行为克隆训练的单一算子能够准确逼近解算子，并能泛化到未见任务、分布外环境以及不同数量的任务观测。我们进一步展示了神经算子架构中的分支-主干结构可实现对新任务的高效且灵活的适应。我们提出了从轻量更新到全网络微调的结构化适应策略，在不同数据和计算环境下均表现出强劲性能。最后，我们引入了元训练的算子变体，以优化初始化以适应小样本任务。该方法能在有限数据情况下快速完成任务适应，并持续优于流行的元学习基线。总之，我们的结果表明，神经算子为多任务控制与适应提供了一个统一且高效的框架。

## Abstract
Neural operator methods have emerged as powerful tools for learning mappings between infinite-dimensional function spaces, yet their potential in optimal control remains largely unexplored. We focus on multi-task control problems, whose solution is a mapping from task description (e.g., cost or dynamics functions) to optimal control law (e.g., feedback policy). We approximate these solution operators using a permutation-invariant neural operator architecture. Across a range of parametric optimal control environments and a locomotion benchmark, a single operator trained via behavioral cloning accurately approximates the solution operator and generalizes to unseen tasks, out-of-distribution settings, and varying amounts of task observations. We further show that the branch-trunk structure of our neural operator architecture enables efficient and flexible adaptation to new tasks. We develop structured adaptation strategies ranging from lightweight updates to full-network fine-tuning, achieving strong performance across different data and compute settings. Finally, we introduce meta-trained operator variants that optimize the initialization for few-shot adaptation. These methods enable rapid task adaptation with limited data and consistently outperform a popular meta-learning baseline. Together, our results demonstrate that neural operators provide a unified and efficient framework for multi-task control and adaptation.

## 一、核心问题与研究背景

- **研究动机**：现有的神经算子（Neural Operator, NO）能学习函数空间之间的映射，在偏微分方程等科学计算中表现优异，但在 **最优控制（Optimal Control）和多任务控制（Multi-task Control）** 场景下仍鲜有应用。  
- **问题本质**：多任务控制可形式化为一个函数空间映射——从任务定义函数（如代价函数或动力学参数）到最优反馈策略。传统方法或需要为每个任务单独训练控制器，或通过低维任务嵌入进行近似，存在泛化性不足、对任务数量敏感等问题。  
- **研究目标**：建立一个统一框架，用神经算子近似“解算子”，实现从任务描述到最优控制律的泛化学习与快速适应，从而在不同任务或分布外环境中高效迁移。

## 二、方法论与技术路线

### 1. 总体思想
- 将多任务控制建模为：  
  \[
  S: \text{任务定义函数} \to \text{最优控制策略}
  \]
  其中输入和输出均属于无限维函数空间。
- 使用 **SetONet** 架构（基于 DeepONet 扩展），学习该函数空间映射。

### 2. 神经算子架构
- **分支（Branch）网络**：接收任务的函数样本（context set），采用置换不变的 MLP + 聚合结构（mean pooling → post-MLP），得到任务相关系数 {cₖ}。
- **主干（Trunk）网络**：生成输出空间的基函数 {bₖ(y)}，与分支系数内积得到预测控制输出。
- 近似算子公式：
  \[
  T_\theta(\ell)(y) = \sum_{k=1}^{p} c_k(\ell) b_k(y)
  \]
- 训练目标为行为克隆（Behavioral Cloning）：
  \[
  L_{BC} = \frac{1}{m}\sum_j \|T_\theta(\ell)(x_j) - \pi^*(x_j)\|^2
  \]
  旨在监督学习专家控制策略。

### 3. 任务适应策略
- **分支-主干分离的微调方式**：
  - **全网络微调（SetONet-FT）**：更新所有参数。
  - **仅分支微调（Full-Branch）**：固定主干，只更新任务系数生成部分。
  - **最后一层微调（Last-Branch / Last-Trunk / Last-Both）**：轻量化更新。
- **成本反馈适应**：在无专家示例条件下，直接最小化控制目标函数的轨迹成本（有动力学模型时可端到端反传）。
- **元训练（Meta-training）**：
  - **SetONet-Meta**：内循环仅更新分支参数，提高数据效率。
  - **SetONet-Meta-Full**：内循环同时更新分支与主干，实现更强适应能力。
  - 采用 MAML 式双层优化目标：  
    - 内环：针对任务支持集执行一次梯度更新。
    - 外环：根据更新后的模型在查询集上的表现反向传播元梯度。

## 六、主要结论与发现

- **神经算子可有效近似控制解算子**：预训练的单一算子能从任务上下文预测最优控制律。
