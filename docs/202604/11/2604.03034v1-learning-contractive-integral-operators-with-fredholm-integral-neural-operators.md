## 摘要
我们推广了弗雷德霍姆神经网络（Fredholm Neural Networks）的框架，以学习来自任意维度二类弗雷德霍姆积分方程（Fredholm Integral Equations, FIEs）中的非扩张性积分算子。我们首先提出了用于FIE的弗雷德霍姆积分神经算子（Fredholm Integral Neural Operators，FREDINOs），并证明其是线性和非线性积分算子及相应解算子的通用逼近器。此外，我们还证明了所学习的算子保证是收缩的，从而严格满足不动点迭代收敛所需的数学条件。最后，我们展示了如何通过边界积分方程（Boundary Integral Equation, BIE）形式，利用FREDINOs学习非线性椭圆型偏微分方程（PDEs）的解算子。我们通过多个基准问题对所提方法进行数值验证，包括任意维度的线性和非线性FIE，以及二维非线性椭圆型PDE。基于定制的数学与数值分析理论，FREDINOs具有高精度逼近和可解释特性，使其非常适用于科学机器学习与数值分析计算。

## Abstract
We generalize the framework of Fredholm Neural Networks, to learn non-expansive integral operators arising in Fredholm Integral Equations (FIEs) of the second kind in arbitrary dimensions. We first present the proposed Fredholm Integral Neural Operators (FREDINOs), for FIEs and prove that they are universal approximators of linear and non-linear integral operators and corresponding solution operators. We furthermore prove that the learned operators are guaranteed to be contractive, thereby strictly satisfying the mathematical property required for the convergence of the fixed point scheme. Finally, we also demonstrate how FREDINOs can be used to learn the solution operator of non-linear elliptic PDEs, via a Boundary Integral Equation (BIE) formulation. We assess the proposed methodology numerically, via several benchmark problems: linear and non-linear FIEs in arbitrary dimensions, as well as a non-linear elliptic PDE in 2D. Built on tailored mathematical/numerical analysis theory, FREDINOs offer high-accuracy approximations and interpretable schemes, making them well suited for scientific machine learning/numerical analysis computations.

## 二、方法论：核心思想与主要技术

### 2.1 方法核心概念
- **FREDINO（Fredholm Integral Neural Operator）**：
  - 是一种以积分形式定义的神经算子，核心在于学习未知的积分核函数 \( K(x, z) \) 及非线性函数 \( G(u) \)。  
  - 该算子保证满足**收缩性条件**（contractive property），即满足 \( \|T f_1 - T f_2\| \le q\|f_1 - f_2\|, q < 1 \)，从而确保解的唯一性与迭代收敛。  

### 2.2 技术细节与数学结构
- **线性与非线性算子定义**：
  \[
  (T f)(x) = \int_D K(x, z) f(z) dz, \quad (T f)(x) = \int_D K(x, z) G(f(z)) dz
  \]
- 通过神经网络对\( K \)与\( G \)建模：\( K_\theta(x,z), G_\vartheta(u) \)。  
- **Fredholm NN映射结构**：
  - **Feedforward FNN**：用于线性FIE，网络层对应于Picard迭代层；
  - **Recurrent FNN**：用于非线性FIE，每层引入递归结构应用非线性\( G(·) \)；
  - 每一层的权重矩阵和偏置项直接由核函数离散形式确定；
  - 深度 \(M\) 即迭代次数，前向传播等价于固定点迭代求解过程。

### 2.3 理论贡献
- 证明了 **FREDINO 是线性/非线性积分算子及其解算子的通用逼近器（Universal Approximator）**。
- 提出**收缩性保持理论证明**：若真实算子为收缩映射，则学习得算子必然也是收缩映射。
- 优化目标为带正则项的最小化问题：
  \[
  \min_{\theta} \|f - F(g,K_\theta)\|_{L_2}^2 + \lambda \|K_\theta\|_{L_2}^2
  \]
  同时引入约束 \( \|K_\theta\|_{L_2}<1 \) 以保证算子收缩。
- 提出**交替最小化算法 (Alternating Minimization)** 用于非线性情形中联合优化 \(K_\theta\) 与 \(G_\vartheta\)。

### 2.4 扩展至 PDE 设置
- 通过**潜在积分方程 (Boundary Integral Equation, BIE)** 将非线性椭圆型 PDE 转化为Fredholm积分形式；
- 提出**Potential Fredholm Neural Network (PFNN)**，将BIE解与双层势函数（double-layer potential）引入最终隐层；
- PDE 学习问题转化为学习修正后的基本解 Φθ(x, y)。

- 手工生成的函数对 \(\{g_i, f_i\}\)，输入 \(g_i(x)\) 为高斯混合、多项式与正弦叠加的随机函数；
- 输出 \(f_i(x)\) 为积分方程精确/数值解生成；
- 高维实验使用 Sobol 低差序列采样（1024 或 2048 个节点）代替常规格点。

### 3.3 benchmark 与评估指标
- 无外部数据集，主要与自身理论一致性对比；
- 评价指标包括**相对 L1、L2、L∞ 误差**及**层间收敛率曲线**；
- 特别通过比较 successive norms \(ρ^{(k)} = ||f^{(k)}-f^{(k-1)}||\) 来验证算子收缩性。

### 3.4 模型配置
- 典型神经网络为3层MLP或RNN模块；
- 离散化节点 N = 500~2048，网络深度 M = 20~40；
- 优化器：Adam + 分段递减学习率，部分实验加 L-BFGS 微调；
- Alternating Minimization 用于非线性算子训练。
