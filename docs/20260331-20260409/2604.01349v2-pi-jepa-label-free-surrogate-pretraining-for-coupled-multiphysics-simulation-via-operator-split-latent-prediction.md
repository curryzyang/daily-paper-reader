## 摘要
油藏模拟工作流程面临一个根本性的数据不对称问题：输入参数场（地质统计学渗透率实现、孔隙度分布）可以任意生成，而现有的神经算子替代模型需要大量昂贵的带标签模拟轨迹数据，无法利用这些无标签结构。我们提出了\textbf{PI-JEPA}（Physics-Informed Joint Embedding Predictive Architecture，物理约束的联合嵌入预测架构），这是一种无需完成任何偏微分方程（PDE）求解的替代模型预训练框架，通过在无标签参数场上进行掩码潜变量预测，并结合每个子算子的PDE残差正则化进行训练。预测器库在结构上与控制方程的Lie–Trotter算子分裂分解保持一致，为每个子过程（压力、饱和度传输、反应）分配独立的受物理约束的潜空间模块，从而在仅使用约100次带标签模拟运行的情况下实现有效微调。在单相Darcy流实验中，PI-JEPA在$N_\ell{=}100$时的误差比FNO低$1.9\times$，比DeepONet低$2.4\times$；在$N_\ell{=}500$时，相较于仅监督训练提升了24\%，这表明无标签替代模型预训练显著降低了多物理场替代模型部署所需的模拟成本。

## Abstract
Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

- **核心问题：数据不对称性（data asymmetry）**

- **现有方法瓶颈**：  
  - 传统神经算子（Neural Operator，如 FNO、DeepONet）依赖大量带标签的（输入→输出）样本进行监督学习。  
  - 即使是物理信息神经网络（PINN/PINO）也要计算 PDE 残差，仍需大量计算资源。  
- **研究动机**：  
  - 期望利用“丰富的无标签输入参数场”进行**无监督或自监督预训练**，从而在少量带标签样本下也能得到准确的仿真代理。  
  - 多物理场方程往往通过算子分裂法（operator splitting）求解，不同物理过程（压力、饱和度、化学反应）在不同时间尺度上演化；现有单体型网络无法有效捕捉这一结构。

## 2. 方法论概述  

### 核心思想：  
提出 **PI-JEPA (Physics-Informed Joint Embedding Predictive Architecture)**——一种**无标签（label-free）替代模型预训练框架**，融合物理约束的自监督学习与算子分裂结构。  

### 关键技术与流程：  
1. **预训练阶段（无标签）**：  
   - 输入：大量随机生成的渗透率或孔隙度场（无需 PDE 求解）。  
   - 使用一种类似 **I-JEPA / V-JEPA** 的“遮掩—预测”（masked latent prediction）结构，通过预测被遮掩区域的潜变量来学习空间相关性。  
   - 采用 **EMA 目标编码器**（Exponential Moving Average）提供稳定表示目标。  
   - 在潜空间层面增加三类损失：
     - **预测损失**：匹配预测潜变量与目标潜变量。
     - **物理残差损失**：为每个算子（如压力、饱和度）计算其 PDE 残差。
     - **VICReg 正则项**：控制潜向量的方差/协方差，防止坍塌。  

2. **算子分裂结构（Operator-Split Prediction Bank）**：  
   - 将整体 PDE 分解为多个子算子（如 Darcy 流中的压力 $L_1$ 与饱和度 $L_2$，反应系统中的 $L_1, L_2, L_3$）。  
   - 每个子算子对应一个独立的潜预测器模块 $g_{\varphi_k}$。  
   - 潜空间演化：$z_c^{(0)} \rightarrow g_{\varphi_1} \rightarrow g_{\varphi_2} \rightarrow \cdots \rightarrow g_{\varphi_K}$。  
   - 各阶段输出通过轻量解码器映射回物理场，用以计算对应的 PDE 残差。  

3. **微调阶段（带标签）**：  
   - 使用少量（如 100 个）全模拟轨迹进行微调。  
   - 在预训练编码器基础上添加预测头（projection + unpatchify + refine），联合优化。  

4. **训练目标公式（概念表达）**：  
   $$
   L = L_{\text{pred}} + \lambda_p \sum_k L_{\text{phys}}^{(k)} + \lambda_r L_{\text{reg}}
   $$
   其中包含预测误差、物理残差项和协方差约束项。

## 6. 主要结论与发现  

- **整体性能**：  
  - 在少样本条件下（\( N_\ell ≤100 \)），PI-JEPA明显超越所有基线。  
  - 在单相 Darcy 流上，误差较 FNO 降低约 **1.8×**，较 DeepONet 降低约 **2.2×**。  
  - 双相 CO₂–水流中，较 FNO 降低 **2.7×**。  
  - ADR 系统中降低约 **1.9×**。  
- **自监督预训练带来 6–14% 的额外提升**，尤其在 \( N_\ell ≥100 \) 时显著。  
- **算子分裂与VICReg为关键组件**：移除后误差分别上升 16% 与 17.7%。  
- **物理残差项效果中性甚至略负**，说明当前近似残差存在离散误差。  
- **PINO ≈ FNO**：监督阶段加入物理项在低样本时几乎无效。

## 7. 方法亮点与贡献  

- **(1) 首个无标签神经算子预训练框架**：可在无需 PDE 求解的情况下获得有效初始表示。

- **(5) 数据效率高**：在仅 100 次带标签仿真下获得 1.8–2.7× 准确度提升。
