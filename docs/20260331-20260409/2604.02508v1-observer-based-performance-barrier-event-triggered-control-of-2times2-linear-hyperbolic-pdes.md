## 摘要
性能障碍事件触发控制（P-ETC）是一种旨在在保持事件触发控制（ETC）所规定系统性能的前提下，增加事件间隔时间（dwell-time）的方法。该方法通过考虑系统的性能残差来实现，性能残差用于衡量系统性能与预设性能之间的差距。这使得李雅普诺夫函数候选项可以不必单调递减。为了确定性能残差，需要系统的全状态信息，因此现有关于 P-ETC 的研究均基于全状态反馈。本文提出了一种在输出反馈下的新的动态性能障碍方法，并配合指数收敛的观测器。我们针对一类具有控制输入与测量位置不重合（anti-collocated）的 $2\times2$ 线性双曲型偏微分方程，研究事件触发边界控制问题。在所提出的 P-ETC 机制下，我们证明了最小停留时间的存在性，并展示了系统解的空间 $L^2$ 范数的全局指数稳定性。最后，通过仿真结果验证了理论分析的正确性。

## Abstract
Performance-barrier event-triggered control (P-ETC) is a methodology implemented to increase the dwell-times between events while still preserving a prescribed performance of the system under event-triggered control (ETC). This is achieved by considering the performance residual of the system, which is a measure of the system performance with respect to the prescribed performance. This allows the Lyapunov function candidate to deviate from decreasing monotonically. In order to determine the performance residual, it is required to know the full-state information, leading to all work related to P-ETC to be under full-state feedback. In this article, we propose a novel dynamic performance-barrier under output feedback with an exponentially convergent observer. We consider event-triggered boundary control of a class of $2\times2$ linear hyperbolic PDEs with anti-collocated measurements with the control input. Under the proposed P-ETC mechanism, we prove the existence of a minimum dwell-time, and show the global exponential stability of the spatial $L^2$ norm of the solution of the system. Simulation results are presented to validate the theoretical claims.

## 一、研究核心与问题背景

- **研究背景**  
  控制理论中，**事件触发控制（Event-Triggered Control, ETC）** 是一种非周期的反馈方式，可在保持系统稳定性的同时减少通信与执行器的更新频率，对于网络化和分布式系统尤其重要。  
  然而，在多数研究中，**性能障碍事件触发控制（Performance-Barrier ETC, P-ETC）** 需要系统的全状态信息以计算性能残差（performance residual），从而限制了其在实际输出受限系统中的应用，尤其是**分布参数系统（PDE 系统）**。  

- **研究动机**  
  针对一类 **$2\times2$ 线性双曲型偏微分方程（hyperbolic PDEs）**，其状态通常只能在边界获取（输出反馈问题），本文旨在：
  1. 克服 P-ETC 需全状态反馈的限制；
  2. 设计一个带有**指数收敛观测器（exponentially convergent observer）** 的输出反馈 P-ETC；
  3. 实现系统稳定性与性能保持的同时，进一步延长事件间隔（dwell-time）。

## 二、方法论：核心思想与技术实现

### 1. 核心思想
- 将传统 P-ETC 引入 **输出反馈控制框架**；
- 采用 **动态性能障碍变量（performance-barrier dynamics）** 与 **观测器估计的状态** 构造触发条件；
- 引入 **估计性能残差（estimated performance residual）** 替代基于全状态的性能残差；
- 使用 **反步法（backstepping method）** 设计边界控制与观测器，实现指数收敛与稳定。

### 2. 数学建模与系统设置
- 被控系统为一对耦合的线性双曲方程：
  \[
  \partial_t u(x,t) = -\lambda_1(x)\partial_x u + c_1(x)v,\quad
  \partial_t v(x,t) = \lambda_2(x)\partial_x v + c_2(x)u
  \]
  边界条件为：
  \[
  u(0,t) = qv(0,t), \quad v(1,t) = \rho u(1,t) + U(t)
  \]
  输出测量为 \( y(t) = v(0,t) \)，存在“**对置（anti-collocated）**”边界感测与控制。

### 3. 观测器与控制器设计
- 构造 **观测器系统**：采用反步变换获得指数收敛的误差方程；
- 控制律以 **“连续控制 + 保持误差 d(t)”** 形式实现：
  \[
  U(t) = U_c(t_j), \quad t \in [t_j, t_{j+1})
  \]
- 性能残差 \(W(t)\) 利用观测状态定义：
  \[
  W(t) = e^{-\gamma t}\rho(t) - \hat{V}(t)
  \]
  其中性能障碍由 \(e^{-\gamma t}\rho(t)\) 描述。

### 4. 事件触发机制
- 动态变量 \(m(t)\) 按分段微分方程演化，事件触发条件为：
  \[
  t_{j+1} = \inf \{ t \geq t_j + \tau \mid m(t) < 0 \}
  \]
  其中 τ 为最小驻留时间（minimum dwell-time），确保不存在 Zeno 现象。
- \(m(t)\) 的演化受 \(W(t)\) 驱动，性能残差为其**延长事件间隔的关键**。

### 5. 稳定性与理论结果
- 利用加权 \(L^2\)-范数构造李雅普诺夫函数候选 \(V(t)\)，并结合动态触发条件证明：
  - 系统解在空间 \(L^2\) 意义下**全局指数稳定（Global Exponential Stability, GES）**；
  - 存在严格正的最小驻留时间 τ；
  - 性能残差项保证在不破坏性能约束的条件下延长触发间隔。
