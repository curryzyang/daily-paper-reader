## 摘要
本文研究了一类含有 Volterra 型记忆项的一维非自治退化抛物方程的记忆型零可控性问题。所考虑的扩散算子既包括散度形式也包括非散度形式，并且在边界处可能表现出弱或强退化，同时扩散系数显式依赖于时间。由于记忆效应的存在，传统的零可控性无法满足需求，因此引入了一种更强的概念，要求状态量和累积记忆同时消失。为解决此问题，我们在加权空间中建立了适用于非自治退化算子的新的 Carleman 估计。记忆项在 Carleman 框架中被视为低阶扰动来处理。这些估计导出了合适的可观测性不等式，从而在适当的结构条件下证明了记忆型零可控性。本文还讨论了双边界退化及控制区域移动情形的推广。

## Abstract
This paper studies the memory-type null controllability of a class of one-dimensional non-autonomous degenerate parabolic equations with Volterra-type memory terms. The diffusion operator is considered in both divergence and non-divergence forms and may exhibit weak or strong degeneracy at the boundary, while the diffusion coefficient depends explicitly on time. Due to the presence of memory effects, classical null controllability is insufficient, and a stronger notion requiring the vanishing of both the state and the accumulated memory is introduced. To address this problem, we establish new Carleman estimates adapted to non-autonomous degenerate operators in weighted spaces. The memory term is handled as a lower-order perturbation within the Carleman framework. These estimates yield suitable observability inequalities, which allow us to prove memory-type null controllability under appropriate structural conditions. Extensions to cases with double boundary degeneracy and moving control regions are also discussed.

## 二、方法论与关键技术

### 1. 问题建模
系统形式为：
\[
u_t - A_i(t)u + \int_0^t M(t,s)u(s)\,ds = f(t,x)\chi_{\omega(t)}(x),
\]
其中：
- \(A_i(t)\)：时间相关的退化扩散算子，分两种形式：
  - 非散度形式：\(A_1(t)u = b(t)a(x)u_{xx}\)；
  - 散度形式：\(A_2(t)u = b(t)(a(x)u_x)_x\)；
- \(a(x)\)：在边界\(x_0\in\{0,1\}\)处可能弱退化（WD）或强退化（SD）；
- \(b(t)\)：时间权重，正且光滑；
- \(M(t,s)\)：记忆核（Volterra 型内积控制）。

### 2. 控制目标  
- 寻找控制 \(f\)，使得：
  \[
  u(T,x)=0,\quad \text{且}\quad \int_0^T f_M(T-s)u(s,x)\,ds=0,
  \]
  即同时令终态与积累的记忆分量为零。

### 3. 分析框架  
- **Carleman估计（Carleman estimate）**：  
  是证明可观测性与可控性的重要工具。  
  本文在加权空间中构造时间与空间权共同作用的权函数，处理**非自治系数b(t)**对权结构的扰动。
- **记忆项处理：**  
  记忆算子被看作低阶扰动项，在Carleman不等式中 absorb 到主项中。  
- **Hilbert Uniqueness Method（HUM）：**  
  通过对偶系统的可观测性不等式推得可控性结论。
- **关键条件：**
  - 边界退化率 \(K_{x_0}<2\)：是确保可控性成立的临界条件；
  - 移动控制区域 \(\omega(t)\)：必须在时间区间内扫过整个空间，以恢复固定区域下观测性失效的问题。

## 六、主要结论与发现

- **主要定理：**  
  一维非自治退化抛物方程在条件 \(K<2\) 下实现记忆型零可控性。  
- **关键发现：**  
  - 记忆项不能简单被消除，必须联动控制状态与记忆核。  
  - 若控制区域固定且非全局，则系统不可控；通过**移动控制**可恢复可观测性与可控性。  
  - Carleman估计框架可自然吸收时间依赖性与记忆效应，保持可观测性不等式的成立。  
  - 边界退化与记忆耦合下的可控性分析可以扩展至双边退化情形。
