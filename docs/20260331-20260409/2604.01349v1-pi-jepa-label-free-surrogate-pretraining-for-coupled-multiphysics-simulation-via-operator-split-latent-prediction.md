## 摘要
储层模拟工作流程面临一种基本的数据不对称性：输入参数场（地质统计渗透率实现、孔隙度分布）可以任意生成大量数据，而现有的神经算子替代模型需要庞大的、昂贵标注的模拟轨迹语料库，无法利用这些未标注的结构。我们提出了\textbf{PI-JEPA}（Physics-Informed Joint Embedding Predictive Architecture，物理信息联合嵌入预测架构），这是一种无需完成任何 PDE 求解的替代模型预训练框架，其通过在未标注的参数场上执行掩码潜空间预测，并在每个子算子 PDE 残差下施加正则化来进行训练。预测器库在结构上与控制方程的 Lie–Trotter 算子分裂分解相对应，为每个子过程（压力、饱和度传输、反应）分配独立的物理约束潜空间模块，从而实现仅需约 100 次标注模拟运行的微调。在单相 Darcy 流实验中，PI-JEPA 在 $N_\ell{=}100$ 时的误差比 FNO 低 $1.9\times$，比 DeepONet 低 $2.4\times$；在 $N_\ell{=}500$ 时相比纯监督训练提升 24%，表明无标注替代模型预训练显著减少了多物理替代模型部署所需的模拟预算。

## Abstract
Reservoir simulation workflows face a fundamental data asymmetry: input parameter fields (geostatistical permeability realizations, porosity distributions) are free to generate in arbitrary quantities, yet existing neural operator surrogates require large corpora of expensive labeled simulation trajectories and cannot exploit this unlabeled structure. We introduce \textbf{PI-JEPA} (Physics-Informed Joint Embedding Predictive Architecture), a surrogate pretraining framework that trains \emph{without any completed PDE solves}, using masked latent prediction on unlabeled parameter fields under per-sub-operator PDE residual regularization. The predictor bank is structurally aligned with the Lie--Trotter operator-splitting decomposition of the governing equations, dedicating a separate physics-constrained latent module to each sub-process (pressure, saturation transport, reaction), enabling fine-tuning with as few as 100 labeled simulation runs. On single-phase Darcy flow, PI-JEPA achieves $1.9\times$ lower error than FNO and $2.4\times$ lower error than DeepONet at $N_\ell{=}100$, with 24\% improvement over supervised-only training at $N_\ell{=}500$, demonstrating that label-free surrogate pretraining substantially reduces the simulation budget required for multiphysics surrogate deployment.

## 一、研究问题与背景

- **核心问题**：  
  储层及多物理场数值模拟通常依赖求解耦合偏微分方程（如达西流、反应-扩散系统），每次高保真模拟成本高昂（需数小时到数天）。然而输入参数字段（如渗透率、孔隙度）可以廉价生成。  
  当前神经算子（如 FNO、DeepONet）需大量带标签数据（输入-输出 PDE 解对）训练，无法利用这些免费但未标注的数据，导致**数据瓶颈与训练成本高**。

- **研究动机**：  
  作者指出存在明显的“数据非对称性”：
  - *标签昂贵*：每一次完整PDE求解都需要昂贵的数值求解器。
  - *输入丰富*：地质统计模型可轻易生成成千上万个参数场。  
  若能仅依赖无标签数据预训练，再用少量标注样本微调，则可极大降低模拟预算。

- **总体目标**：  
  提出一种**无标签自监督预训练框架**，利用物理结构（PDE分裂算子）和潜空间预测学习，实现高精度、低监督的多物理场替代模型。

## 二、方法论

### 1. 核心思想

- 提出 **PI-JEPA (Physics-Informed Joint Embedding Predictive Architecture)**：
  - 借鉴 LeCun 提出的 *Joint Embedding Predictive Architectures (JEPA)* 概念；
  - 结合物理约束（PDE 残差正则化）；
  - 与算子分裂（Operator Splitting，如 Lie–Trotter）结构对齐；
  - 通过自监督掩蔽预测学习潜空间动态，而**不需要任何完整 PDE 求解**。

### 2. 架构概览

- **主要组成：**
  1. **上下文编码器 fθ** 与 **目标编码器 fξ（EMA平均更新）**：生成潜表示；
  2. **潜空间预测器组 {gϕk}**：每个对应一个物理子算子（如压力、饱和度、反应）；
  3. **PDE 残差正则化项**：对每个子算子分别计算物理残差；
  4. **VICReg 协方差正则**：防止潜空间坍缩；
  5. **掩蔽预测策略**：在潜空间预测目标嵌入而非像素重建。

- **算子对应结构**：
  - 数值求解中常用 **Lie–Trotter 分裂**：
    \[
    S(Δt) ≈ L_K(Δt) ∘ … ∘ L_1(Δt)
    \]
    每个 Lₖ 为物理子过程；
  - PI-JEPA 对应每个子算子设计一个 **预测器 gϕₖ** 并独立施加物理残差约束。

### 3. 损失函数

\[
L = L_{pred} + λ_p \sum_k L_{phys}^{(k)} + λ_r L_{reg}
\]

- **L_pred**：潜空间预测误差（与互信息掩蔽预测类似）；  
- **L_phys**：每子算子 PDE 残差（利用轻量解码器计算）；  
- **L_reg**：VICReg 正则，保持潜变量方差与解耦。  
- **EMA更新**：目标编码器参数 ξ ← τξ + (1−τ)θ，用于稳定目标嵌入。

### 4. 算子对应示例

- **压力子算子 L₁**：达西流中的椭圆方程；
- **饱和度子算子 L₂**：双相流中的超曲线输运方程；
- **反应子算子 L₃**：化学反应守恒约束。

### 5. 扩展与细节

- **输入掩蔽策略**：采用时空块（spatiotemporal block）掩蔽，在时间上预测下一时刻的空间邻域潜表示；
- **推理方式**：在潜空间进行自回归滚动（autoregressive rollout），并通过解码器重建物理量；
- **优势**：分离快（椭圆型）与慢（超曲线型、反应型）子过程，减少任务干扰。
