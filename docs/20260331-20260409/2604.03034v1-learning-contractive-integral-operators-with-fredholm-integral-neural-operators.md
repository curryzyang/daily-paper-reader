## 摘要
我们推广了 Fredholm 神经网络的框架，以学习在任意维度中出现于第二类 Fredholm 积分方程（FIEs）中的非扩张积分算子。我们首先提出了用于 FIEs 的 Fredholm 积分神经算子（FREDINOs），并证明它们是线性和非线性积分算子及其对应解算子的通用逼近器。此外，我们证明所学习到的算子保证是收缩的，从而严格满足固定点迭代法收敛所需的数学性质。最后，我们还展示了如何通过边界积分方程（BIE）形式，将 FREDINOs 用于非线性椭圆偏微分方程的解算子学习。我们通过多个基准问题对所提出的方法进行数值评估：任意维度下的线性和非线性 FIEs，以及二维非线性椭圆 PDE。基于定制的数学与数值分析理论，FREDINOs 提供高精度的逼近和可解释的方案，使其非常适合科学机器学习和数值分析计算。

## Abstract
We generalize the framework of Fredholm Neural Networks, to learn non-expansive integral operators arising in Fredholm Integral Equations (FIEs) of the second kind in arbitrary dimensions. We first present the proposed Fredholm Integral Neural Operators (FREDINOs), for FIEs and prove that they are universal approximators of linear and non-linear integral operators and corresponding solution operators. We furthermore prove that the learned operators are guaranteed to be contractive, thereby strictly satisfying the mathematical property required for the convergence of the fixed point scheme. Finally, we also demonstrate how FREDINOs can be used to learn the solution operator of non-linear elliptic PDEs, via a Boundary Integral Equation (BIE) formulation. We assess the proposed methodology numerically, via several benchmark problems: linear and non-linear FIEs in arbitrary dimensions, as well as a non-linear elliptic PDE in 2D. Built on tailored mathematical/numerical analysis theory, FREDINOs offer high-accuracy approximations and interpretable schemes, making them well suited for scientific machine learning/numerical analysis computations.

## 二、方法论与算法框架

### 1. 核心思想  
- 基于作者此前的 **Fredholm Neural Network (FNN)** 框架，将神经网络层与 **Krasnosel’skii–Mann** 或 **Picard 固定点迭代**进行等价映射。  
- 每一层网络即对应一次固定点迭代，从而实现对积分算子**逐层近似与收敛控制**。  
- 提出的 **Fredholm Integral Neural Operator (FREDINO)** 可以学习：
  1. 未知的积分核函数 \( K(x,z) \)，  
  2. 非线性函数 \( G(f(z)) \)（可选，用于非线性情形）。  

### 2. 核心技术要点  
- **Fredholm Integral Equation (FIE)** 两种形式：  
  - 线性：\( f(x) = g(x) + \int_D K(x,z)f(z)dz \)  
  - 非线性：\( f(x) = g(x) + \int_D K(x,z)G(f(z))dz \)
- **构造神经网络结构**：
  - 线性情形：前馈型 Fredholm NN（Feedforward FNN）。
  - 非线性情形：引入递归结构（Recurrent Fredholm NN），在每层中内嵌非线性 \(G(\cdot)\)。  
- **关键数学结果**：
  - 证明 FREDINO 是**线性与非线性积分算子的通用逼近器（universal approximator）**。  
  - 并且学习得到的算子满足 **\(\|T\| < 1\)** 的收缩约束，保证固定点迭代收敛。  
- **训练目标**：
  - 最小化损失函数  
    \[
    L(\theta) = \|f - F(\cdot; g, K_\theta, G_\vartheta)\|^2 + \lambda(\|K_\theta\|^2 + \|G_\vartheta\|^2)
    \]
    并施加收缩约束 \( \|K_\theta\|_{L^2(D\times D)} < 1 \)。  
  - 优化过程中等价于求解受约束的 Tikhonov 正则化问题。  

### 3. 优化与训练方案  
- 非线性情形采用**交替最小化（Alternating Minimization）**算法，分别优化 \(K_\theta\) 与 \(G_\vartheta\)。  
- 每一次迭代包含：
  1. 固定 \(G_\vartheta\) 训练 \(K_\theta\)。
  2. 固定 \(K_\theta\) 训练 \(G_\vartheta\)。
  3. 可选联合微调阶段（使用 L-BFGS）。  

- 对于 PDE，应对 2D 非线性椭圆方程，通过 **Potential Fredholm Neural Network (PFNN)** 建模双层势积分方程（Boundary Integral Equation, BIE），实现在数值稳定条件下学习 PDE 解算算子。

### 2. 数据与输入函数
- 输入函数族 \(g(x)\)：由多项式 + 高斯混合 + 正弦信号组成的随机函数集合（参照 DeepONet 测试集生成方式）。  
- 每类问题分别生成 200–600 个训练样本，500–2000 测试样本。  
- 使用 **Sobol 序列**进行高维积分的低偏差采样。  

### 3. 模型与对比
- FREDINO 内部网络：
  - 核网络（kernel network）\(K_\theta\)：1–3 层 MLP（64–128 个神经元）。  
  - 非线性网络 \(G_\vartheta\)：2 层 ReLU 或 SiLU 网络。  
  - Fredholm NN 前向传播层数：20–40 层，节点数与积分离散点相同。
- 无直接外部对比方法（论文定位为理论性工作），但与 **Fourier Neural Operator、DeepONet** 在概念层面对比其黑箱特性与本方法的可解释性。
